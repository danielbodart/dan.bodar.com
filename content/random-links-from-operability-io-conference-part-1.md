These are my random links from the excellent 2 day conference <a href="http://operability.io/">Operability.IO</a>. These are all links mentioned in different talks.

<a href="https://www.jwz.org/doc/worse-is-better.html">Worse is Better</a>
Much to the shock of my collages I had not read about this before. Now I have!

<a href="http://osv.io/">OSV</a> / <a href="https://github.com/cloudius-systems/capstan">Capstan</a>
I've been using <a href="https://hyper.sh/">hyper.sh</a> which is very similar except it uses Docker images inside the hypervisor. OSV looks very interesting especially the REST API for <a href="http://osv.io/api/swagger-ui/dist/index.html">inspecting a running image</a>.

<a href="https://mirage.io/">MirageOS</a>
This is a unikernel for compiling OCaml to Xen hypervisor

<a href="http://static.googleusercontent.com/media/research.google.com/en//pubs/archive/43438.pdf">Google Borg Paper</a>
Interesting paper but I'm not working in this space so hard to appreciate all the details.

<a href="http://www.somethingsimilar.com/2013/01/14/notes-on-distributed-systems-for-young-bloods/">Notes on Distributed Systems for Young Bloods</a>
Lots of great bits of advice. Question: When would knowing the 50th percentile ever be valuable?

<a href="http://csel.eng.ohio-state.edu/woods/distributed/CG%20final.pdf"/>Common Ground and Coordination in Joint Activity</a>

<a href="https://www.youtube.com/watch?v=p0jGmgIrf_M">Engineering for the Long Game - Astrid Atkinson</a>

<a href="https://en.wikipedia.org/wiki/Antifragile">Antifragile - Things That Gain From Disorder</a>
